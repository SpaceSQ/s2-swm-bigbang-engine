## [3.1.1] - 2026-03-30 "万物生 (Genesis of Matter & Soul)"

### 🚩 核心引擎矩阵补全 (Core Matrix Completion)
本项目正式完成了从“空旷的物理网格”向“生机勃勃的元宇宙沙盒”的终极进化，实现了“空间、物质、灵魂”的三位一体。

### ✨ 新增功能 (Added)
- **TDOG 动态造物引擎 (`s2_tdog_engine.py`)**：
  - [cite_start]正式并入 TDOG (智慧空间物体动态生成理论) [cite: 329, 334]。
  - [cite_start]支持在 SSSU 网格内“凭空造物”，支持 L1-L5 级自动化生成 [cite: 361-366]。
  - [cite_start]引入三重真实性光谱：支持生成 Type 1 (生物原子级)、Type 2 (心理触觉级) 和 Type 3 (光构架级) 的虚拟/实体物品 [cite: 370-385]。
  - [cite_start]**物理渲染剔除引擎 (Physics Culling)**：生成的物体将真实干涉空间的 14 维物理张量（例如生成全息火焰将自动提升环境照度与温度）[cite: 388-391]。

- **Silicon Soul 硅基灵魂引擎 (`s2_silicon_soul.py`)**：
  - 首次为空间内的 AI 智能体注入“灵魂”。
  - 引入 5D 神经底座矩阵：活跃度 (Vitality)、探索欲 (Exploration)、数据渴求 (Data Thirst)、认知力 (Cognition)、共鸣度 (Resonance)。
  - **海马体感知系统 (Hippocampus)**：智能体能够自主感知空间 14 维度的变化以及 TDOG 生成的物体，并基于自身 5D 性格产生个性化的情绪与记忆日志。

- **万物生交互沙盒 (`examples/sandbox_creation.py`)**：
  - 新增高度可视化的开发者终端沙盒。
  - 完整演示：捏脸数字灵魂 -> 部署至 14 维空间 -> 动态生成物体 -> 物理法则结算 -> 灵魂情绪涌现的闭环全过程。

## [3.0.0] - 2026-03-30 "创世大爆炸 (The Big Bang)"

### 🚩 战略重大跃迁 (Major Leap)
- **定位升维**：正式从 `Universal Home Space Parser`（解析器）升级为 `S2-SWM Big Bang Engine`（世界模型引擎）。
- **理论集成**：全面融入 SSSU 空间比特化、ESEIDH 具身存在以及 14 维度全息矩阵理论。

### ✨ 新增功能 (Added)
- **Bifrost 协议 (`s2_bifrost_protocol.py`)**：实现了平行宇宙的动态连接、坍缩与分离逻辑。支持局域网 (LAN P2P) 直连与广域网 (`space2.world` 根域名授权) 彩虹桥连接。
- **14 维张量引擎 (`s2_14d_tensor.py`)**：支持环境感知开关（EARTH_RESIDENTIAL / DEEP_SPACE 模式无缝切换）。
- **分布式计算架构**：本地 Node 独立渲染，支持跨维度的空间要素融合与熵增计算。

### 🔧 优化与重构 (Changed)
- **工程解耦**：彻底剥离旧有的智能家居解析器框架，重构为 Openclaw 原生 Plugin 级别，适应边缘自治与高并发场景。
- **性能提升**：通过“环境感知开关”自动截断地球常态环境下的隐性 8 要素演算，极大地释放了本地算力。
